# PID制御の基本

PID 制御は「いまどれだけずれているか」「ずれがどれだけ積み上がったか」「どれだけ速く変化しているか」を見ながら操作量を決める考え方です。
モーターの回転数制御や位置制御では、この 3 つの役割を理解すると現場の調整がぐっと読みやすくなります。

## まず見るポイント

- `P` を上げると素早く追従しやすくなりますが、上げすぎると振動しやすくなります。
- `I` を上げると小さなずれを消し込みやすくなりますが、効かせすぎると行き過ぎが増えます。
- `D` を上げるとブレーキのように効き、止まり方が落ち着きやすくなります。

## 実機につなげて考える

実機のモーターでも、基本はこの画面と同じです。
センサで現在値を読み、目標値との差から操作量を決め、その結果として回転数や位置が変わります。

- 回転数制御ではエンコーダやホールセンサの値が現在値になります。
- 位置制御では回転数だけでなく、停止時の振動や慣性も重要になります。
- 外乱ボタンは、坂道や負荷変動のような「急な条件変化」をイメージしてください。
