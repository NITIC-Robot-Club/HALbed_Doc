# モーター回転数制御

回転数制御は「どのくらいの速さで回したいか」を保つ制御です。
搬送機構や車輪制御では、負荷が変わっても回転数を保てることが大切になります。

## 学びどころ

- 慣性があるので、出力を変えても回転数はすぐには変わりません。
- 負荷が増えると、同じ出力でも回転数が落ちやすくなります。
- PID と C++ 制御則を切り替えると、同じモデルに対する考え方の違いが見えます。

## 実機に近い見方

ブラシレスモーターでも DC モーターでも、回転数ループでは「目標 rpm」と「現在 rpm」の差を埋める考え方が基本です。
PWM のデューティ比や電流指令に相当するものが、このシミュレータの操作量 `u` です。
