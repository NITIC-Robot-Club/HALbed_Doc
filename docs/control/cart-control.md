# 台車制御ゲーム

ここでは、目標地点に素早く止める制御をゲーム感覚で学びます。
コードがそのまま挙動に反映されるので、比例だけで動かしたときの弱点や、速度項を入れたときの改善が見えやすくなります。

## 進め方

1. まずはテンプレートのまま実行して、どの程度安定して止まるか見ます。
2. `error` の係数を変えて、追従の速さと振動の変化を確認します。
3. `state.velocity` を使って減速を強め、停止の質を上げます。
4. 外乱を入れても止まれるか試します。
