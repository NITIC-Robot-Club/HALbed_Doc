# 位置制御

位置制御では、目的地に届くだけでなく「行き過ぎずに止まれるか」が重要です。
回転数制御よりも慣性の影響を受けやすいため、速度や加速度も意識して見ると理解しやすくなります。

## ここに注目

- `P` だけでは勢い余って通り過ぎやすくなります。
- `D` は止まる直前のブレーキ役として働きます。
- C++ 制御則を書くと、速度フィードバックの大切さが体感しやすくなります。
