# 位置制御

位置制御では、目的地に届くだけでなく「行き過ぎずに止まれるか」が重要です。
このページでは、エレベータや昇降機構のように上下方向へ動くモデルとして扱い、速度や加速度も見ながら止まり方を確かめます。入力が 0 なら重力で下がり、床の 0 より下へは落ちない条件です。

## ここに注目

- `P` だけでは勢い余って通り過ぎやすくなります。
- `D` は止まる直前のブレーキ役として働きます。
- 上が正、下が負の向きとして見せることで、台車ゲームより「高さで止める」感覚に集中できます。
- C++ 制御則を書くと、速度フィードバックの大切さが体感しやすくなります。
