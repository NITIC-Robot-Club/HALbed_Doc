# PWMとは何か

PWMは、LEDの明るさやモーターの速さを変えるときによく使う信号です。このページでは、PWMがどのようなものかを確認しながら、用途によって設定が変わる理由を見ていきます。

## オンとオフの時間を調整する

PWMは、オンとオフをとても速く繰り返す信号です。1回の繰り返しの中で、オンにしている時間の割合をデューティ比と呼びます。

```text
デューティ比 = オン時間 / 周期
周波数 = 1 / 周期
```

周期が1 msでオン時間が0.25 msなら、デューティ比は25%、周波数は1 kHzです。

## なぜ出力を調整できるのか

GPIOの瞬間的な電圧は、オンかオフのどちらかです。しかしLEDの見え方やモーターへ伝わる平均的なエネルギーは、オン時間の割合に応じて変化します。そのため、アナログ電圧を直接出していなくても、明るさや回転の強さを調整できます。

負荷によっては、FETやモータードライバーなどの駆動回路が必要です。マイコンのGPIOへモーターを直接接続してはいけません。

## タイマーとの関係

STM32では、タイマーの数が`CNT`、1回の長さが`ARR`、オンにする長さが`CCR`です。タイマーがこれらを比べて、PWMのオンとオフを切り替えます。

同じタイマーの複数チャネルは周期を共有することがあります。一方のチャネルだけ周波数を変えようとすると、同じタイマーを使うほかのPWMにも影響する可能性があります。

## 用途ごとの考え方

LEDでは、ちらつきが見えない程度の周波数を選び、デューティ比で明るさを変えます。DCモーターでは、ドライバーの仕様、騒音、損失を考えて周波数を決めます。

RCサーボもPWMに似た周期信号を使いますが、一般に角度はデューティ比よりパルス幅で指定します。そのため、LED用のPWM設定をそのまま流用はできません。

## 分解能と周波数の関係

速く切り替えるほど、明るさや速さを細かく変えるための段階は減りやすくなります。まずは用途に必要な周波数を決めてから、調整の細かさを考えましょう。

## まとめ

PWMは、周期的なデジタル信号のオン時間を変える技術です。周波数とデューティ比を区別し、接続する機器の仕様に合わせて設定します。
